# Repository access

Source repository:

```text
https://github.com/noamglikman/work4u.git
```

Before delivery, grant the customer or examiner read access to the repository.
If the repository remains private, add the reviewer as a collaborator or provide
read access through the organization/team that owns the repository.

The ZIP delivery is self-contained, but repository read access is still required
when source-control access is part of the submission requirements.
