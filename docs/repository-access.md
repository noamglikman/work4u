# Repository access

Source repository:

```text
https://github.com/noamglikman/work4u.git
```

Required access:

- Grant the receiving technical team or examiner read access to the repository.
- If the repository is private, add the reviewer as a collaborator or grant read access through the owning GitHub organization/team.
- Keep the repository available during the installation and review period.

The ZIP package is self-contained, but repository read access is required when source-control access is part of the submission requirements.
